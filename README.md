# MatrixLibrary

## Build and install



Just like:

```
$ cmake [params] PATH/TO/MatrixLibrary
$ make
$ make install
```

### Params

#### BUILD_TARGET

##### iOS

`-DBUILD_TARGET=iOS`

##### macOS

`-DBUILD_TARGET=macOS`

##### win32

`-DBUILD_TARGET=win32`

##### Android

`-DDBUILD_TARGET=Android`
